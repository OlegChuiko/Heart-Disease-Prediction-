# 🫀 Heart Disease Prediction using Deep Learning (PyTorch)

## 📌 Опис проєкту
Цей проєкт присвячений розробці системи машинного навчання для прогнозування наявності серцево-судинних захворювань у пацієнтів на основі їхніх медичних показників. У рамках проєкту було пройдено повний цикл Data Science: від експлораторного аналізу (EDA) та очищення даних до побудови та порівняння кількох моделей класифікації, включаючи глибоку нейронну мережу на базі PyTorch.

## 📊 Дані та Передобробка (Data Preprocessing)
Набір даних містить клінічні показники пацієнтів (вік, стать, тип болю в грудях, артеріальний тиск, рівень холестерину тощо).
В процесі підготовки даних було виконано:
* **Обробку аномалій:** Виявлено медично неможливі нульові значення в колонках `Cholesterol` та `RestingBP`. Їх було замінено на медіанні значення для уникнення спотворення роботи алгоритмів.
* **Кодування (One-Hot Encoding):** Категоріальні текстові ознаки (наприклад, тип болю `ASY`, `ATA`) перетворено у числовий формат.
* **Масштабування (StandardScaler):** Всі числові ознаки приведено до єдиного масштабу, що є критичною вимогою для коректного навчання нейронних мереж.

## 🤖 Моделі машинного навчання
Для порівняння ефективності було побудовано три моделі:
1. **Logistic Regression** (Baseline / Базова модель).
2. **Multi-Layer Perceptron (MLP)** (Scikit-Learn).
3. **Deep Neural Network** (PyTorch) — кастомна архітектура з використанням `Batch Normalization`, `Dropout` для боротьби з перенавчанням та `L2 Регуляризації`.

## 📈 Результати
Оскільки у медичних завданнях найважливішим показником є **Recall** (здатність моделі знаходити всіх хворих і мінімізувати хибнонегативні результати - False Negatives), моделі оцінювалися саме з акцентом на цю метрику.

* **Логістична регресія:** Recall = 82.35% (18 False Negatives)
* **MLP (Scikit-Learn):** Recall = 86.27% (14 False Negatives)
* **PyTorch DNN:** Recall = **88.24%** (всього 12 False Negatives)

**Висновок:** Кастомна глибока нейронна мережа на PyTorch показала найкращі результати, значно зменшивши кількість критичних помилок. Ця модель може бути використана як інструмент "другої думки" (second opinion) для лікарів-кардіологів при первинному скринінгу пацієнтів.
